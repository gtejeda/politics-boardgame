# Recursos

Este directorio contiene recursos visuales y de producción para el juego.

---

## Estructura del Directorio

```
assets/
├── icons/          # Iconografía del juego
├── cards/          # Plantillas y diseños de cartas
├── board/          # Archivos de diseño del tablero
├── tokens/         # Diseños de fichas/marcadores
├── box/            # Arte de caja y empaque
├── rulebook/       # Archivos de diseño del libro de reglas
└── reference/      # Materiales de referencia visual
```

### Principios Visuales
- Codificación de colores neutral (evitar rojo=malo, verde=bueno)
- Abstracto sobre realista (reducir asociaciones del mundo real)
- Iconografía clara sobre texto donde sea posible
- Diseño accesible (amigable para daltónicos, fuentes legibles)

### Consideraciones de Estilo
- Modernista/limpio vs. texturizado/ilustrado
- Definición de paleta de colores
- Sistema de iconografía
- Selección de tipografía

## Convención de Nombres de Archivos

```
[elemento]-[variante]-[versión].[extensión]
```

Ejemplos:
- `policy-card-template-v1.svg`
- `budget-icon-filled-v2.png`
- `board-layout-4player-v3.pdf`
